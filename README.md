# To-Do List Application – Backend API

This project is the Backend for a Full-Stack To-Do List Application.

It is built using Node.js, Express.js, and PostgreSQL and RESTful API design.

## Tech Stack

Node.js

Express.js

PostgreSQL

Sequelize ORM

dotenv (Environment Variables)

CORS

##  Objective

Build a RESTful API that supports:

Create a Task

Get All Tasks

Update a Task (Edit title / Mark as completed)
